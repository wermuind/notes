# Стандартная библиотека

<dfn>C++ Standard Library</dfn> (стандартная библиотека C++) — набор классов
и функций, написанных на самом языке и являющихся частью стандарта C++.

<dfn>Standard Template Library / STL</dfn> (стандартная библиотека шаблонов) —
подмножество стандартной библиотеки, набор согласованных обобщённых алгоритмов
и контейнеров.
