# Стандарты JavaScript

[Таблица совместимости](http://kangax.github.io/compat-table/es2016plus/)

## Нововведения

### ES5 (2009)

+ JSON
+ строгий режим
+ `String.trim()`, `Array.isArray()`
+ `for..in`, `for..of`

### ES2015

+ `let` и `const`, параметры по умолчанию
+ `Array.find()`, `Array.findIndex()`

### ES2016

+ `Array.includes()`
+ оператор возведения в степень

### ES2017

+ асинхронные функции (async-await)
+ `String.padStart()`, `String.padEnd()`
+ `Object.values()`, `Object.entries()`
+ разделяемая память и атомарные операции

### ES2018

+ операторы rest и spread
+ `Promise.finally()`, асинхронные итерации
+ новые возможности для регулярных выражений

### ES2019

+ `Object.fromEntries()`
+ `Array.flat()`, `Array.flatMap()`
+ `String.matchAll()`, `String.trimStart()`, `String.trimEnd()`
+ `Symbol.Description`
+ `catch` без переменной
+ улучшения `Array.sort()`
+ обновленный `Function.toString()`
+ `BigInt`
+ динамический импорт

### ES2021

+ `String.replaceAll()`
+ приватные методы класса и приватные аксессоры
+ `WeakRef` и `FinalizationRegistry`
+ `Promise.any()` и `AggregateError`
+ операторы логического присваивания `&&=` , `||=` и `??=`

### ES2022

+ `Array.at()`
+ `Object.hasOwn()`
+ `Error.cause` — указание какая ошибка вызвала другую
+ await вне функции в модулях

### ES2023

+ `Array.findLast()`, `Array.findLastIndex()`
+ `Array.with()`, `Array.toSorted()`, `Array.toReversed()`, `Array.toSliced()`
