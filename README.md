# Notes

Notes — статический сайт (Docusaurus), на котором собираются заметки
на различные темы, с которыми, как правило, приходилось детально разбираться.

## Содержимое

```text
notes/
├─ cpp           # заметки по C++
│  ├─ language   #   описание языка
│  └─ std        #   стандартная библиотека
├─ js            # заметки по JavaScript
├─ db            # базы данных
├─ compiler      # теория компиляторов
│  ├─ lexer      #   лексический анализ
│  └─ parser     #   синтаксический анализ
├─ low-level     # низкоуровневое программирование
└─ bookmarks     # закладки
```
