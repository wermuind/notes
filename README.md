# Notes

Notes — статический сайт (движок Docusaurus), на котором собираются заметки
на различные темы, с которыми доводилось разбираться достаточно глубоко,
чтобы написать об этом. По сути, это — живая документация моих интересов
и пути обучения программированию.

## Структура

```text
notes/
├─ cpp           # заметки по C++
│  ├─ language   #   описание языка
│  └─ std        #   стандартная библиотека
├─ js            # заметки по JavaScript
├─ db            # базы данных
├─ compiler      # теория компиляции
│  ├─ lexer      #   лексический анализ
│  └─ parser     #   синтаксический анализ
├─ low-level     # низкоуровневое программирование
└─ bookmarks     # закладки
```
