# Notes

Notes — это статический сайт, на котором собираются заметки на различные темы,
с которыми доводилось разбираться достаточно глубоко, чтобы написать об этом.
По сути, это открытая база знаний, живая документация моих интересов и пути
обучения программированию.

## Каталоги

```text
notes/
├─ cpp           # заметки по C++
│  ├─ language   #   описание языка
│  └─ std        #   стандартная библиотека
├─ js            # заметки по JavaScript
├─ db            # базы данных
├─ compiler      # теория компиляции
│  ├─ lexer      #   лексический анализ
│  └─ parser     #   синтаксический анализ
├─ low-level     # низкоуровневое программирование
└─ bookmarks     # закладки
```

## Стиль

#### Типографика:
+ длинное тире (`—`) вместо среднего (`–`) или дефиса (`-`)
+ неразрывный пробел необязателен

#### Markdown:
+ жирный текст: `**b**` вместо `__b__`
+ курсив: `*i*` вместо `_i_`
+ неупорядоченный список верхнего уровня через `+`, вложенный через `-`
+ для заголовков с прямой ссылкой задаётся ASCII идентификатор
